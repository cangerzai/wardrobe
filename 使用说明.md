# 使用说明

## 开发环境配置

### 1. 安装微信开发者工具
- 下载并安装最新版本的微信开发者工具
- 访问：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

### 2. 创建小程序项目
1. 打开微信开发者工具
2. 选择"小程序"项目类型
3. 填写项目信息：
   - 项目名称：电子衣橱
   - 目录：选择本项目文件夹
   - AppID：填写你的小程序AppID（测试可以使用测试号）
4. 点击"新建"

### 3. 配置AppID
在 `project.config.json` 文件中，将 `appid` 字段修改为你的小程序AppID。

### 4. 配置云开发（可选）
如果需要使用云开发功能：
1. 在微信开发者工具中开通云开发
2. 创建云环境
3. 在 `app.js` 中配置云环境ID

## 集成真实风格迁移API

### 方案1：使用云函数

1. 在云开发控制台创建云函数 `styleTransfer`
2. 在云函数中调用风格迁移API（如使用Python的CartoonGAN）
3. 修改 `utils/styleTransfer.js` 中的 `transferToCartoon` 函数：

```javascript
async function transferToCartoon(imagePath) {
  return await callCloudFunction(imagePath)
}
```

### 方案2：使用第三方API

修改 `utils/styleTransfer.js` 中的 `callThirdPartyAPI` 函数，配置：
- API地址
- API密钥
- 请求格式

### 方案3：使用腾讯云图像处理

1. 注册腾讯云账号
2. 开通图像处理服务
3. 获取API密钥
4. 在云函数中调用腾讯云API

## 常见问题

### Q: Canvas绘制不显示？
A: 确保调用了 `ctx.draw()` 方法，Canvas需要手动触发绘制。

### Q: 图片上传失败？
A: 检查小程序权限设置，确保已授权相册和相机权限。

### Q: 风格迁移不工作？
A: 当前使用模拟处理，需要集成真实的风格迁移API。参考"集成真实风格迁移API"部分。

### Q: 数据丢失？
A: 当前使用本地存储，清除小程序数据会丢失。建议集成云数据库。
