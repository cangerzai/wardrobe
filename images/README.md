# 图片资源目录

此目录用于存放小程序所需的图片资源。

## 必需文件

### TabBar图标（必需）
请准备以下图标文件，建议尺寸：81x81px

- `home.png` - 首页未选中图标
- `home-active.png` - 首页选中图标
- `wardrobe.png` - 衣橱未选中图标
- `wardrobe-active.png` - 衣橱选中图标
- `discover.png` - 发现未选中图标
- `discover-active.png` - 发现选中图标
- `profile.png` - 我的未选中图标
- `profile-active.png` - 我的选中图标

### 默认头像（可选）
- `default-avatar.png` - 默认用户头像，建议尺寸：200x200px

## 图标设计建议

- 使用简洁的线条图标
- 选中和未选中状态使用不同颜色区分
- 保持图标风格统一
- 建议使用SVG格式转换为PNG

## 获取图标资源

1. **免费图标库**：
   - IconFont: https://www.iconfont.cn/
   - Flaticon: https://www.flaticon.com/
   - Icons8: https://icons8.com/

2. **设计工具**：
   - Figma
   - Sketch
   - Adobe Illustrator

3. **在线工具**：
   - Canva
   - 稿定设计

## 注意事项

- 如果图标文件不存在，TabBar会显示默认样式
- 图片格式建议使用PNG，支持透明背景
- 图片大小建议控制在50KB以内
- 可以使用图片压缩工具优化文件大小
